# Product Context: ArchCrafter2

## Description
A GTK3-based desktop customization suite for Arch Linux, specializing in Openbox window management and visual aesthetics.

## Primary Users
Arch Linux desktop users and Openbox enthusiasts who prefer a GUI for environment customization.

## Main Goal
To provide a centralized, efficient, and visual interface for skinning and configuring the desktop, eliminating the need for manual configuration file editing.

## Main Features
- **Wallpaper Management:** Async scanning of system and custom directories, color palette extraction, and ImageMagick-powered colorization.
- **Window Theme Management:** Real-time Openbox theme detection from system and user directories with instant application via `openbox --reconfigure`.
- **Planned Features:** GTK theme switching, icon/cursor management, and direct Openbox XML configuration tweaks.
